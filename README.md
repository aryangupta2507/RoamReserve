# RoamReserve – MERN Hotel Booking Platform

🌐 Explore, Book, and Host – A full-stack hotel booking and accommodation management platform built using the MERN stack.  
This project enables users to discover properties, make reservations, and host accommodations through an interactive and responsive web interface.

---

## 🚀 Overview

RoamReserve is a full-stack web application designed to simplify hotel and property booking.  
The platform provides seamless user interaction for both travellers and hosts through role-based dashboards and real-time booking features.

Users can:
- Explore and search accommodations
- Book properties with secure authentication
- Host and manage listings
- Control bookings and properties through personalized dashboards

---

## ✨ Features

- 🔐 Secure Authentication with JWT
- 🏨 Property Booking and Hosting System
- 📊 Personalized User Dashboards
- 🗺️ Geolocation-based Property Discovery using Leaflet Maps
- 📱 Fully Responsive UI Design
- 🧾 Booking and Listing Management
- ⚙️ Role-Based Access Control
